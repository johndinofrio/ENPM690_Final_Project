# ENPM690_Final_Project

## Dependencies

The following dependencies must be installed. There are a lot of them, and it might be a pain to install some of them.

1. python3.5 or above 
2. numpy 
3. collections
4. tensorflow
5. random
6. skimage
7. gym
8. retro
